# wtf is this?
an over-engineered webpage that displays my own personal splits of Autechre's live material. Why? Stop aksing questions

## Looks cool, i wanna make one of these for myself
Well you can! All of the content (the table content, split information, even the set headers) are all data-driven. They read entries from a few text files to dynamically generate everything.
You can pull the repo and edit the text files, and have your own version of this webpage but with your own split data! You don't even need to make it about Autechre specifically!
You don't need to touch any other files other than the 4 text files within the repo under `src/data`
(Although, that's obviously what the stylizing of this webpage was built for).
